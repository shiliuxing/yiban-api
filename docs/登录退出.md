### 登录退出接口

#### 管理员登录

`POST` `http://localhost:4568/admin/auth`

接受参数：

| 参数 | 意义 | 备注 |
| ------ | ------ | ------ |
| username | 管理员账号 | 必填 |
| password | 管理员密码 | 必填 |

返回格式：

| 参数 | 意义 | 备注 |
| ------ | ------ | ------ |
| code | 登录结果 | code 为 0 时登录成功，其他情况为登录失败 |
| msg | 提示信息 | 登录成功时为”登录成功“，失败时为登录失败原因 |

#### 管理员退出
